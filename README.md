## 研究主线一：A 股因子研究

### 数据工程

- 数据源：akshare，沪深300成分股，前复权日线
- 数据清洗：复权处理（送股/派息/配股）、停牌缺失值填充、异常值 MAD 截断、全样本避免幸存者偏差

### 因子有效性检验

对动量、反转、换手率三类因子进行截面 IC 检验：

| 因子 | IC 均值 | ICIR | IC>0 比例 | 结论 |
|------|---------|------|-----------|------|
| 动量（20日） | -0.0023 | -0.011 | 50.6% | 无效 |
| 反转（5日） | 0.0062 | 0.032 | 49.6% | 无效 |
| 换手率 | 0.0290 | 0.109 | 55.1% | 弱有效 |
| 三因子等权合成 | 0.0328 | 0.224 | 58.3% | 弱有效 |

IC 合格标准：均值 > 0.05，ICIR > 0.5，IC>0 比例 > 55%。三因子均未达到单独使用的门槛。

### 策略回测与收益归因

| 策略版本 | 累计收益 | Sharpe | 最大回撤 |
|----------|---------|--------|---------|
| 三因子合成基础版 | +8.56% | 0.24 | -22.73% |
| + 200日均线择时 + 单日止损 | +16.75% | 0.43 | -15.06% |
| 沪深300基准 | +10.17% | — | — |

**收益归因结论**

通过分离择时与选股贡献：

- 纯择时（200日均线，满仓沪深300成分股）：累计 +65%，Sharpe 1.15
- 纯选股（无风控，三因子选前20%）：累计 +8.56%，Sharpe 0.24
- 结论：当前超额收益完全来源于择时模块，因子选股贡献为零，需进一步改进因子质量

## 研究主线二：加密货币择时策略

### 数据基础设施

| 数据类型 | 来源 | 覆盖范围 |
|----------|------|---------|
| BTC/USDT 日线 | OKX API | 2023.01 至今，1183条 |
| BTC/USDT 小时线 | OKX API | 2023.01 至今，28385条 |
| 资金费率 | 币安 API | 2019.09 至今，7174条 |
| 恐贪指数 | Alternative.me | 2020.10 至今，2000条 |
| 链上活跃地址 | CoinMetrics | 2020.10 至今 |

### 策略迭代路径

**基础版：MACD 择时**

| 参数版本 | 累计收益 | Sharpe | 最大回撤 |
|----------|---------|--------|---------|
| 默认参数 (12,26,9) | +135% | 1.00 | -32% |
| 网格搜索最优 (16,35,9) | +339% | 1.61 | -21% |

**加入情绪过滤**

| 过滤因子 | 逻辑 | 累计收益 | Sharpe |
|----------|------|---------|--------|
| 资金费率 > 0.03% 空仓 | 市场过热退出 | +310% | 1.17 |
| 恐贪指数 > 80 空仓 | 极度贪婪退出 | +423% | 1.70 |

**加入止损模块**

| 止损阈值 | 累计收益 | 最大回撤 | 年均交易次数 |
|----------|---------|---------|------------|
| 1% | +1335% | -5% | 32次（过拟合风险）|
| 3% | +443% | -15% | 11次 |
| 无止损 | +274% | -22% | 0次 |

选择 3% 止损：兼顾收益提升与交易成本可控。

**扣除手续费后（最终版本）**

```
策略：MACD(16,35,9) + 恐贪指数(<80) + 3% 止损
每年交易：21次  手续费：每次 0.1%  总成本：~7%
扣费后累计收益：+384%  Sharpe：1.70  最大回撤：-16%
```

### Walk-Forward 验证

将数据切成4段，每段用前70%训练参数，后30%验证：

| 段 | 最优参数 | 训练Sharpe | 验证Sharpe |
|----|---------|-----------|-----------|
| 第1段 | (16,35,9) | 2.99 | -0.65 |
| 第2段 | (16,35,12) | 2.90 | -0.18 |
| 第3段 | (20,26,12) | 1.73 | 3.22 |
| 第4段 | (8,21,7) | 0.42 | 0.09 |
| **平均** | — | — | **0.62** |

验证集平均 Sharpe 0.62，策略有一定有效性但存在参数不稳定性，需持续监控。

### 因子有效性汇总

| 因子 | 测试结果 | 是否纳入策略 |
|------|---------|------------|
| MACD 趋势跟踪 | Sharpe 1.61，核心信号 | ✅ 是 |
| 恐贪指数 | 单独弱，组合提升 Sharpe 至 1.70 | ✅ 是（过滤器）|
| 资金费率 | 单独 Sharpe 1.17，组合提升有限 | 备选 |
| 成交量因子 | 单独无效，叠加过度过滤 | ❌ 否 |
| 链上活跃地址 | IC = -0.01，本轮周期机构ETF驱动导致失效 | ❌ 否 |

### 进阶方向探索

| 方向 | 结果 | 核心原因 |
|------|------|---------|
| 多周期共振（日线+小时线） | 累计 +199%，跑输 | 双重过滤使持仓时间降至28%，错过主升浪 |
| 做空策略 | 累计 +216%，跑输 | BTC 长期向上偏差，空头拖累整体表现 |
| 三档仓位管理 | 累计 +215%，跑输 | BTC 强趋势资产，全仓进出优于比例仓位 |
| 波动率自适应仓位 | 累计 +153%，跑输 | 高波动期往往是主升行情，减仓得不偿失 |
| 小时线 MACD(150,400,20) | 扣费前 +682%，扣费后 +439% | 每年102次交易，手续费侵蚀大量超额收益 |

## 研究主线三：机器学习信号

### 特征工程

构建13个技术特征，涵盖动量、MACD、波动率、成交量、RSI、布林带位置、恐贪指数。

### 实验结果

| 预测目标 | 数据频率 | 样本量 | 准确率 | 结论 |
|----------|---------|--------|--------|------|
| 未来5天价格涨跌方向 | 日线 | 1,183 | 51% | 接近随机，无效 |
| 未来10天价格涨跌方向 | 日线 | 1,183 | 45% | 低于随机，无效 |
| 未来24小时价格涨跌方向 | 小时线 | 28,217 | 47% | 接近随机，无效 |
| **未来24小时波动率高低** | **小时线** | **28,217** | **67%** | **有效** |

### 波动率预测策略

基于滚动训练（10折）的波动率预测信号，构建风险规避策略：

- ML 风险预测（仅波动率信号）：累计 +200%，Sharpe 1.10，跑赢买入持有
- MACD + ML 组合（扣费前）：累计 +178%，Sharpe 1.34
- MACD + ML 组合（扣费后）：累计 +48%，手续费侵蚀 45%

**结论：ML 波动率预测在日频有应用价值，小时频因交易成本问题暂不适合实盘部署。**

## 核心研究结论

**1. 简单策略的鲁棒性优于复杂策略**
经过系统测试，MACD + 情绪过滤 + 止损的简单组合在风险调整后收益上优于所有复杂化尝试。

**2. 收益归因是策略评估的必要环节**
A股策略超额收益100%来自择时模块，选股因子贡献为零。未做归因的策略优化方向会严重偏离。

**3. 交易成本决定策略的可用频率下限**
日线策略（21次/年）：手续费可忽略不计。小时线策略（100次+/年）：手续费消灭超额收益。这是策略频率选择的核心约束。

**4. Walk-Forward 验证是防过拟合的必要门槛**
参数优化后的样本内 Sharpe 普遍虚高（最高达2.99），验证集 Sharpe 才反映真实水平（平均0.62）。

**5. 资产特性决定策略形态**
BTC 是强趋势、长期向上偏差资产，适合全仓趋势跟踪；不适合仓位管理、做空策略。A股适合截面因子横向比较选股。
